С чего начать работу над проектом с препроцессорами

  • Установить/обновить node.js последнюю версию, если возникают ошибки скачать здесь

  • Устанавливаем Grant

    1. Из консоли запускаем команду для глобальной установки пакетов grunt и grunt-cli. 
      npm i -g grunt grunt-cli

  • Устанавливаем Галп

    1. Из консоли запускаем команду для глобальной установки пакета gulp. 
      npm i -g gulp gulp-cli
    2. В корневой папке проекта создаем файл gulpfile.js

  • Устанавливаем плагин для Галпа

    1. Из консоли запускаем команду для установки пакета (в данном случае - это gulp-sass) gulp-sass для компиляции SASS-файлов.
      2. npm i --save-dev gulp-sass
