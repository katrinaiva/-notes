С чего начать работу над проектом с препроцессорами

  1. Установить/обновить node.js последнюю версию, если возникают ошибки скачать здесь

  2. Устанавливаем Grant

    • Из консоли запускаем команду для глобальной установки пакетов grunt и grunt-cli. 
      npm i -g grunt grunt-cli

  3. Устанавливаем Галп

    • Из консоли запускаем команду для глобальной установки пакета gulp. 
      npm i -g gulp gulp-cli
    • В корневой папке проекта создаем файл gulpfile.js

  4. Устанавливаем плагин для Галпа

    • Из консоли запускаем команду для установки пакета (в данном случае - это gulp-sass) gulp-sass для компиляции SASS-файлов.
      • npm i --save-dev gulp-sass
